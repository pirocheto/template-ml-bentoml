### Iris classifier service

It is a classification service that uses the features of iris plant to return a type of iris plant.

It uses the following features:

- sepal length in cm
- sepal width in cm
- petal length in cm
- petal width in cm

The returned class can be:

- Iris Setosa
- Iris Versicolour
- Iris Virginica
